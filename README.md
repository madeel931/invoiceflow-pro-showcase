# InvoiceFlow Pro by ADii Labs

**InvoiceFlow Pro** is an offline-first Flutter invoice management app designed for freelancers, contractors, service providers, and small businesses.

This repository is a public showcase for the app.
The full source code is private and may be available separately for purchase, customization, or client projects.

## Overview

InvoiceFlow Pro helps small businesses manage invoices, customers, products, business profile details, invoice statuses, PDF invoice export, and local backup/restore.

The app is built with a local-first approach, meaning business data is stored on the device and does not require a cloud subscription for core usage.

## Core Features

* Customer management
* Product and service catalog
* Invoice creation
* Invoice status tracking
* Dashboard and business overview
* Business profile setup
* Multi-currency support
* PDF invoice export
* Local backup and restore
* English, Urdu, and Arabic user interface
* Offline-first local data storage

## Tech Stack

* Flutter
* Dart
* Clean Architecture
* Cubit / Bloc state management
* Isar local database
* GetIt dependency injection
* GoRouter navigation
* PDF generation
* Local-first app architecture

## Target Users

InvoiceFlow Pro is designed for:

* Freelancers

## App Direction

The goal of InvoiceFlow Pro is to provide a simple, practical, offline-first invoicing system for users who need fast invoice creation, customer records, product/service management, PDF export, and local data control.
